========================================================================
             영화 시맨틱 검색 엔진 프로젝트 기반 기술 명세서 (Tech.txt)
========================================================================

이 프로젝트는 대규모 언어 모델(LLM)의 생성 능력과 임베딩 모델(Embedding Model)의 
의미론적 분석력을 결합하여, 서버 비용을 최소화하면서 고성능 검색 기능을 제공하는 
'하이브리드 로컬 시맨틱 검색 시스템'입니다.

1. 데이터 수집 및 구조화 기술 (Data Collection & Structuring)
------------------------------------------------------------------------
* 사용 모델: Gemini 3.1 Flash Lite (gemini-3.1-flash-lite)
* 핵심 기술: Structured Outputs (구조화된 출력)
  - Python Pydantic 라이브러리를 활용하여 출력 스키마(Schema)를 강제 지정.
  - API 반환값의 JSON 포맷 유효성 및 데이터 정합성(제목, 연도, 배우, 줄거리)을 100% 보장.
* 수집 알고리즘: 중복 검사 루프 (Incremental Loop)
  - 기존 수집 데이터를 제외 대상으로 지정하여 API를 중복 호출하지 않도록 자동 요청.
  - 카테고리당 정확히 고유 데이터 100개씩(총 300개) 수집될 때까지 반복 확보.

2. 의미론적 벡터 임베딩 기술 (Semantic Embedding)
------------------------------------------------------------------------
* 사용 모델: Gemini Embedding v2 (gemini-embedding-2)
* 출력 차원: 3,072차원 고정밀 밀집 벡터 (Dense Vector)
* 문서 가공: 영화 메타데이터(제목, 개봉연도, 배우)와 서사(줄거리)를 한 문단으로 
  결합하여 문맥적 의미가 보존된 문서(Document) 형태를 제작해 임베딩 진행.
* 최적화: 증분 임베딩 (Incremental Caching)
  - 기존 임베딩 데이터베이스(movies_embeddings.json)를 사전에 로드.
  - 신규 데이터 추가 시, 이미 생성된 벡터 연산은 재사용하고 새 영화만 선별하여 
    임베딩 연산을 수행하므로 API 할당량(Quota) 및 대기 시간 극단적 절약.

3. 로컬 유사도 분석 엔진 (Local Search Engine)
------------------------------------------------------------------------
* 임베딩 변환: 사용자의 자연어 검색 질문을 gemini-embedding-2 모델을 사용하여 
  실시간 3,072차원의 쿼리 벡터로 변환.
* 유사도 알고리즘: 코사인 유사도 (Cosine Similarity)
  - 수식: Similarity = A · B / (||A|| * ||B||)
  - 두 벡터 간의 사잇각 코사인 값을 산출하여 의미론적 유사성을 -1 ~ 1 범위(실제 0 ~ 1)로 계측.
  - Python의 'NumPy' 라이브러리(np.dot 및 np.linalg.norm)를 활용하여 고속 벡터 연산 처리.
* 카테고리 필터링: 전체 300개의 데이터셋 중 활성화된 카테고리(한국 영화, 해외 영화, 
  일본 애니메이션) 100개에 대해서만 연산을 격리 수행하여 검색 신속성 및 연관성 극대화 (Top 3 노출).

4. 초경량 API 백엔드 서버 (Backend API)
------------------------------------------------------------------------
* 구동 환경: Python 3.11+
* 기술 스택: Python 표준 라이브러리 (http.server 및 socketserver)
  - 추가 프레임워크(Flask, FastAPI 등) 없이 작동하여 무거운 패키지 배포 의존성 제거.
* 제공 API:
  - GET /api/movies?category={category}: 해당 카테고리의 100개 영화 목록 로드 (벡터 필드 제외로 트래픽 최적화).
  - GET /api/search?q={query}&category={category}: 시맨틱 검색 결과 Top 3 반환.

5. 프리미엄 웹 인터페이스 (Frontend UI/UX)
------------------------------------------------------------------------
* 기술 스택: Vanilla HTML5, CSS3, Javascript (ES6+)
* 디자인 테마: 다크 모드 기반의 글래스모피즘 (Glassmorphism)
  - backdrop-filter: blur() 속성을 이용한 반투명 유리 질감 카드 디자인.
  - HSL 보색 대비 및 Indigo-Pink 계열의 네온 그라데이션 글로우 배경 레이아웃.
* 인터랙션:
  - 카테고리 전환 시 즉각적인 탭 토글 및 카드 페이드인(Fade-in) 키프레임 애니메이션 렌더링.
  - 비동기 통신(Fetch API) 기반 화면 갱신으로 새로고침 없는 매끄러운 검색 환경 제공.
